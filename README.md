
# Mind Valut Game



## Technologies Used

- HTML
- CSS
- JavaScript

## Description

MindVault is a memory challenge game where players memorize number sequences and type them back correctly before time runs out. The game has three difficulty levels and tests memory, concentration, speed, and accuracy.

## User Stories
•	As a user, I want to Choose a difficulty level.
•	As a user, I want to Memorize a random number sequence.
•	As a user, I want to Enter the sequence after it disappears.
•	As a user, I want to Receive feedback for correct or incorrect answers.
•	As a user, I want to Track score and lives.
•	As a user, I want to Advance through increasingly difficult levels.
•	As a user, I want to Restart and play again.
